## Types

### Entry

#### Example:

```js
{
  key: 'some key',
  value: 'some value'
}
```

### Raw Entry

#### Example:

```js
{
  key: 'some key',
  value: 'some value',
  createdAt: 1644086231045
}
```

### Key

Can be **any** type, except `null` or `undefined`, which may cause unexpected issues.

### Value

Can be **any** type, except `null` or `undefined`, which may cause unexpected issues.
